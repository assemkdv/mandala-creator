# Mandala Creator 

An interactive web app for creating beautiful, symmetrical mandala patterns in your browser.

## Live Demo

🔗 [Try it here](https://assemkdv.github.io/mandala-creator)

## Features

- Draw symmetrical mandala patterns in real time
- Adjustable brush size and color
- Symmetry controls for different pattern styles

## Built With

- HTML
- CSS
- JavaScript (Canvas API)

## Getting Started

To run locally, simply clone the repo and open `index.html` in your browser:

```bash
git clone https://github.com/assemkdv/mandala-creator.git
cd mandala-creator
open index.html
```
